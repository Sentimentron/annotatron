Just one more thing...

annotatron couldn't find an OpenFaaS connection set up, we need one to continue.

To configure this, open annotatron/settings.py and add a line like this:

            OPENFAAS_URL = "http://localhost:8080/"

Then, restart the annotatron core with docker-compose

            docker-compose restart annotatron-service

Or, pass AN_OPENFAAS_URL in the environment when you do docker-compose up.

For more information about configuring OpenFaaS, see the Wiki on Github.
